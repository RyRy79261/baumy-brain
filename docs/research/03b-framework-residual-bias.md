Residual-bias / gap list. Not a rewrite — just what still leans or is still missing.

## (1) Priors still smuggled into "neutral bedrock" (§I)

- **N2** — "cue-dependent, *not a readout of absolute trace strength*" over-reads. Availability≠accessibility is neutral; the "not absolute trace strength" gloss is a Tulving-side denial that the signal-detection/strength camp (Wixted) explicitly rejects — strength theory holds retrieval IS partly a readout of a graded strength signal. A rival is being defeated inside a bedrock line.
- **N5** — "separating **fast-specific from slow-integrated** learning" is CLS vocabulary presented as biology-agnostic bedrock. Not all interference mitigations map to a fast/slow *separation* (weight-regularization/EWC, replay, orthogonalization, scale don't cleanly). The doc quarantines *necessity* to R9 but still states the "separation" *benefit* in one school's terms. The neutral kernel is only "catastrophic interference exists and is mitigable."
- **N1** — claim is neutral, but the enumerated dissociation set (recency/primacy, span-vs-LTM) is the modal-store paradigm's own canonical demo list; the *selection* is lineage-tinted even where the proposition survives.

## (2) Schools / data still missing

- **Retrieved-context models (TCM→CMR, Polyn/Kahana)** — only TCM appears as a scattered R5 aside; as a full modern account of recency, organization, and the recency "scalar," it belongs seated in R4/R3/R10, not footnoted.
- **Oberauer** WM and **Ericsson–Kintsch long-term working memory** — WM is represented only by Baddeley + Cowan; the concentric/embedded and LT-WM rivals are absent.
- **Memory-augmented neural nets (NTM, DNC, Memory Networks, slot/key-value memory) and vector-DB RAG** — these *literally implement addressable read/write*, so R5's "address lookup = strawman nobody holds" is factually wrong for the AI literature the doc claims to cover. Biggest single omission.
- **Concrete agent-memory systems** (MemGPT/Letta, Generative Agents, RAG-vs-long-context evals) — given the stated LLM/agent scope, the AI leg is thin relative to cog-psych/neuro; N4's possession-denial cites no agent-memory eval data.
- **KG/retrieval literature** — one of the four named literatures is nearly unrepresented: link-prediction/KG-completion (TransE etc.), dense-vs-sparse retrieval, entity resolution. It survives only via PageRank/Hopfield analogies.
- **RL experience replay / prioritized replay (Schaul)** — an AI-native "importance scalar over stored items" (bears on R4) and a fast/slow analogue (R9); absent.
- **Synaptic tagging-and-capture / neuromodulatory consolidation (Frey & Morris)** — the molecular salience-gating mechanism directly relevant to R8 and the T8 "salience-gated single-shot write."
- **Sparse Distributed Memory (Kanerva) and VSA/holographic reps (Plate)** — associative-memory rivals on point for N8/R6.
- **Anti-representational / enactive-ecological memory** (beyond Bartlett) — a genuine "no stored trace at all" pole that would contest the trace talk in N2; constructivism is seated but still trace-adjacent.

## (3) Framework-relative questions with still-uneven weighting

- **R5** — the third pole is now seated, but the *address-lookup* pole is dismissed as "strawman null nobody holds" while NTM/DNC/slot-memory/vector-DBs instantiate exactly it. One pole defeated by mislabeling, not by evidence.
- **R1** — naming **state-trace analysis** as "the actual adjudicating instrument" is not neutral: state-trace is the single-substrate camp's preferred tool and tends to collapse dissociations to one dimension; its own critics (monotonicity assumptions, reversed-association debates) go uncited. The "decider" tilts toward the process side the doc admits it leans on here.
- **R10** — post-correction the interference/convergence side now carries three citations (McGeoch, Underwood, Jenkins–Dallenbach) against one for decay (Hardt); the old decay thumb looks over-corrected into a convergence thumb.
- **R4** — importance anchors were broadened to affect/value/prediction-error but omit the AI-native RL prioritized-replay anchor, leaving the "importance" side still cog-psych-weighted.

## (4) Is the bedrock truly school-independent?

Mostly yes, and the per-item pointers to relative cousins are the doc's strongest feature — but not fully: **N2** and **N5** (above) still carry one school's denial/vocabulary. Deeper structural residue: the entire **T1–T8 scaffold and the R-question titles remain in store/systems vocabulary**, which the doc *names* as the original bias but does **not** dismantle — naming ≠ neutralizing, so R1–R11 are still organized around the very framing they critique. And "framework-neutral" as a *category* presupposes that phenomena can be stated ontology-free, which the enactive/anti-representational pole (missing, §2) would itself deny — so the bedrock is school-independent only relative to the schools actually seated.
