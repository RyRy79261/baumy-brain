## Multi-store / modal model
- school: Systems / multi-store (architecture)
- commits: Memory is a flow through structurally distinct stores (sensory register -> short-term store -> long-term store) with transfer governed by control processes; boxes-and-arrows are real.
- proponents: Atkinson & Shiffrin 1968; Waugh & Norman
- diverges: This is the frame the corpus tacitly adopts and then only partially defends. A modal-model proponent would say T1 undersells it (the store ontology is treated as 'genuinely live contestedness' rather than the default), while a critic notes the corpus admits the scheme 'was the von-Neumann computer metaphor' yet still lets the AI fast/slow mapping ride on it — importing the metaphor twice and calling the echo convergence.

## Multicomponent working memory
- school: Systems / working-memory (architecture)
- commits: Short-term maintenance is a dedicated multi-part system (phonological loop, visuospatial sketchpad, episodic buffer, central executive), not merely activated long-term memory.
- proponents: Baddeley & Hitch 1974; Baddeley 2000
- diverges: Would object that T1 files the short-term/long-term cut under 'contested' and leans toward Cowan's activated-LTM reading; the multicomponent camp holds the dissociations (dual-task, word-length, irrelevant-speech) warrant a genuinely separate store, so the corpus's tilt toward 'continuum' is a thumb against structural WM.

## Declarative / nondeclarative taxonomy
- school: Cognitive neuroscience of memory systems
- commits: Lesion double-dissociations carve memory into declarative (hippocampal, flexible, conscious) vs a heterogeneous nondeclarative bundle (procedural, priming, conditioning); the divisions are biologically real.
- proponents: Squire; Cohen & Squire 1980
- diverges: Broadly congenial to the corpus (T1 durable core), but a strict Squire proponent would flag that the corpus lets the four-box working/episodic/semantic/procedural scheme borrow this lesion warrant, when the lesion data licenses only coarse declarative/nondeclarative plurality — the corpus's own T1 note concedes this, then the AI mapping quietly re-inflates it.

## SPI (Serial-Parallel-Independent) model
- school: Tulving systems / process-dependency
- commits: Encoding into episodic memory is serially dependent on semantic memory, storage is parallel, and retrieval is independent — systems are real but asymmetrically coupled, not modular islands.
- proponents: Tulving 1995
- diverges: Would say any agent taxonomy that treats episodic and semantic as independently readable stores (as AI 'episodic logs' + 'fact graphs' do) violates the serial encoding dependency; the corpus's T1-B leniency toward 'episodic-like' agent memory ignores that even Tulving's own architecture forbids free-standing episodic storage.

## Embedded-processes model
- school: Single-system / activated-LTM
- commits: There is no separate short-term box; working memory is the currently activated subset of long-term memory plus a narrow focus of attention. Memory is one store with graded activation.
- proponents: Cowan 1988, 1999, 2019
- diverges: A direct challenge to the whole store-partition the corpus builds on. Cowan would argue the corpus's entire 'memory is plural (systems)' backbone (T1) and its AI store-per-function mapping presuppose discrete containers that the human data does not support — the sidelining of embedded-processes is the corpus's structural bias.

## Single-system / signal-detection continuity
- school: Unitary-memory / state-trace
- commits: Apparent multiple systems reduce to one continuous memory-strength dimension read out by different decision criteria; remember/know and recollection/familiarity are one signal, not two stores.
- proponents: Dunn 2004; Wixted & Squire; Berry, Shanks & Henson
- diverges: Would call the corpus's plurality core (T1) and its episodic-vs-semantic AI mapping an artifact of criterion-setting, not architecture. The corpus grants this camp a paragraph (T1-C) but never lets it threaten the systems framing it needs for the AI analogy — a selective quarantine of the strongest deflationary rival.

## Connectionist / PDP distributed memory
- school: Connectionism / subsymbolic
- commits: Memories are superposed distributed patterns of weights in one network; there are no discrete traces, no symbolic triples, and dissociations emerge from a single mechanism without separate stores.
- proponents: Rumelhart & McClelland 1986; McClelland; Van Orden; Plaut
- diverges: The most systematically sidelined structural rival. It would reject the corpus's core representational vocabulary — 'traces,' 'facts/triples,' 'encoding is distillation into stored items' (T2), 'edges/weights that mean something' (T3) — as symbolic-storage assumptions. Single-mechanism dissociation models (Dunn & Kirsner) show the corpus's plurality-from-dissociation inference (T1) is non-unique; the corpus notes this once and moves on.

## Complementary Learning Systems (CLS)
- school: Computational neuroscience / dual-system consolidation
- commits: Interference forces a two-timescale architecture: a fast sparse hippocampal encoder plus a slow interleaving neocortex; this is near-necessary, not one design among many.
- proponents: McClelland, McNaughton & O'Reilly 1995; Kumaran, Hassabis & McClelland
- diverges: The corpus (T5) leans on CLS as a 'structural answer / near-theorem,' then walks it back to sufficiency under attack. A CLS partisan would say the walk-back over-credits the single-network scale rival; an anti-CLS reader says the corpus still smuggles CLS in as the default fast/slow justification for AI reflection pipelines — either way the framing is contested and the corpus picks a side implicitly.

## Scale-as-substitute single-network
- school: Deep-learning continual-learning
- commits: A single large network's gradients orthogonalize with scale, buying pattern-separation-like protection from interference without any second store — one substrate suffices.
- proponents: Ramasesh, Lewkowycz & Dyer 2022; Mirzadeh et al.
- diverges: Would say the corpus's CLS-flavored T5 (and its endorsement of fast/slow reflection) inflates a sufficiency result into necessity; the corpus concedes this in the weakened tier but its top-line 'two-timescale is the structural answer' still reads as a thumb toward dual-store consolidation over emergent single-system protection.

## Multiple Trace Theory / trace-transformation
- school: Hippocampal memory neuroscience
- commits: Episodic detail always re-engages the hippocampus; consolidation multiplies and transforms traces rather than transferring them to a hippocampus-independent cortical store — direction of 'consolidation as transfer' is wrong.
- proponents: Nadel & Moscovitch 1997; Moscovitch; Winocur
- diverges: Directly contradicts the corpus's 'replay transforms episodes into gist that migrates to slow store' (T5-B) directional frame. An MTT proponent says the corpus adopts standard-consolidation directionality as if settled; the meta-review itself flags this (T5-Claim-D under-weights MTT directionality), i.e., the corpus knows it tilted and left it in.

## Standard consolidation + fast cortical schema learning
- school: Systems-consolidation neuroscience
- commits: Memory transfers from hippocampus to neocortex over time, BUT a matching prior schema lets neocortex learn in one shot, collapsing the slow-cortex premise.
- proponents: Squire & Alvarez 1995; Tse et al. 2007
- diverges: The Tse result is the corpus's own admitted 'crack' in T5. A schema-learning proponent would say the corpus files fast cortical learning as a 'refinement' rather than a refutation of the slow-neocortex premise its AI consolidation analogy depends on — an unfalsifiability move (schema-consistency defined post hoc by observed learning rate).

## Engram-cell theory
- school: Molecular/optogenetic memory neuroscience
- commits: A specific memory is stored in a sparse, physically identifiable ensemble of cells reactivatable on cue; 'silent engrams' show retrieval failure vs storage failure are physically distinguishable.
- proponents: Tonegawa; Josselyn & Tonegawa 2020; Ryan et al.
- diverges: Cuts against the corpus's T8-B verdict that 'where is a fact stored' is ill-posed / a convergence. Engram work says storage locus IS identifiable and dissociable from retrieval — so the corpus's lean toward distributed-non-localizability (borrowed to defend the LLM analogy) sidelines a large, prize-winning localizationist program.

## Levels-of-processing
- school: Process / depth-of-encoding
- commits: Retention is a byproduct of depth of processing at encoding, not transfer between stores; memory is what processing leaves behind, so there is no separate structural architecture to describe.
- proponents: Craik & Lockhart 1972; Craik & Tulving 1975
- diverges: A process theorist would say the corpus's structure-first framing (stores, traces, graphs) mislocates the phenomenon: there is no 'thing' stored to weight or decay, only processing traces. The corpus dismisses depth as 'circular' (T2-C) — which an LoP proponent calls the standard strawman that ignores the reformulation as transfer-appropriate processing.

## Transfer-appropriate processing
- school: Process / encoding-retrieval interaction
- commits: Performance depends on the match between encoding operations and retrieval operations, not on any absolute trace strength or storage location.
- proponents: Morris, Bransford & Franks 1977; Roediger
- diverges: Would say the corpus's weighting backbone (T3: recency x importance x relevance as properties OF a stored item) is a category error — retrievability is a relation between two processes, not a scalar attached to a trace. The corpus's whole 'edge weight' discourse (T3-D) presupposes the storage-strength ontology TAP denies.
